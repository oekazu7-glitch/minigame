# Super Komatsu Man

Phaser 3 で作った、一人用のボンバーマン風アクションゲームです。

## 遊び方

- 移動: `WASD` / 矢印キー
- 爆弾設置: `Space`
- リスタート: `R`
- 一時停止: `P`
- デバッグ用ステージ移動: `Shift + 1`〜`Shift + 9` / `Shift + 0` (`0` はステージ10)

スマートフォンでは画面下の方向ボタンと `BOMB` ボタンを使えます。

敵を全滅させ、壊せる壁に隠された出口へ入るとステージクリアです。全10ステージあります。

ステージクリア後は、SCOREを使って「爆弾追加」「火力アップ」「スピードアップ」から
1つを選択できます。購入せずに次へ進むこともできます。

## 敵リスト

| 敵 | LIFE | 行動 |
| --- | ---: | --- |
| ノーマル | 1 | 壁や爆弾にぶつかるまで直進し、ぶつかると方向転換します。 |
| チェイサー | 1 | マス中央で進路を判断し、プレイヤーへ近づく方向を選びます。 |
| ゴースト | 1 | 壊せる壁を通過できます。移動方向はランダムです。 |
| パックンマン | 2 | 青いパックマン風の敵です。爆弾を見つけると近づき、接触した爆弾を消します。 |
| シュバリエ | 2 | 騎士風の敵です。前方2マス以内にプレイヤーがいると、槍を表示して速度2倍で突進します。 |
| バクダソ | 1 | 爆弾型の敵です。約7秒ごとに爆弾化して爆発し、その後元に戻ります。爆弾化中は爆風で誘爆しますが倒れません。 |
| ケダマ | 1 | 爆風を受けると燃え上がって高速追跡を始め、約5秒後に爆発します。燃焼中は倒せません。 |
| テポドン | 1 | 前方にプレイヤーを見つけると、ロケット形態で高速突進します。壁にぶつかるかプレイヤーへ到達すると広範囲に爆発します。 |

## ローカル起動

静的ファイルだけで動作します。簡易サーバーを起動して `index.html` を開いてください。

```powershell
python -m http.server 8000
```

ブラウザで <http://localhost:8000> を開きます。

## GitHub Pages

1. このフォルダを GitHub リポジトリへ push します。
2. リポジトリの `Settings > Pages` を開きます。
3. `Build and deployment` の Source に `GitHub Actions` を選びます。

`main` ブランチへの push 時に `.github/workflows/pages.yml` がサイトを公開します。
