# Super Komatsu Man

Phaser 3で作った、一人用のボンバーマン風アクションゲームです。
敵を全滅させ、壊せる壁に隠された出口へ入るとステージクリアです。

## 現在の仕様

- 全20ステージ、5エリア構成です。
- 各エリアは4ステージずつです。
- ステージクリア後、SCOREを使って3択ショップからパワーアップを購入できます。
- 敵撃破時にSCOREがその場に表示されます。
- 短時間に同時撃破するとコンボ倍率が上がります。
- OP、タイトル、ステージ開始、エリア別BGM、爆発音があります。
- ステージ9にJKsoldierが1体登場します。

## 操作

| 操作 | キー |
| --- | --- |
| 移動 | `WASD` / 矢印キー |
| 爆弾設置 | `Space` |
| アクション | `E` |
| リスタート | `R` |
| 一時停止 | `P` |
| OPスキップ | `Enter` 長押し |
| デバッグコンソール | `Ctrl + Alt + Shift` |

`E` は所持パワーアップに応じて、グローブの爆弾投げ、またはリモコン爆弾の起爆に使います。

スマートフォンでは画面下の方向ボタンと `BOMB` ボタンで操作できます。

## ゲーム進行

1. ステージ内の敵を全滅させます。
2. 壊せる壁のどこかに隠された出口を探します。
3. 敵が0体の状態で出口に入るとステージクリアです。
4. クリア後にショップが開きます。
5. SCOREが足りればパワーアップを購入できます。購入せず次へ進むこともできます。

## パワーアップ

| レア度 | 出現率 | パワーアップ |
| --- | ---: | --- |
| COMMON | 70% | BOMB +1、FIRE +1、SPEED +1、LIFE +1 |
| RARE | 25% | BOMB +2、FIRE +2、SPEED +2、HEART、LIFE +2 |
| EPIC | 5% | BOMB +3、FIRE +3、SPEED +3、PUNCH、GLOVE、WALL PASS、PIERCE BOMB、REMOTE、LIFE +3 |

### レベル系

- BOMB: 同時設置数が増えます。最大8です。
- FIRE: 爆風距離が伸びます。最大8です。
- SPEED: 移動速度が上がります。最大8段階です。
- LIFE: 残機が即時増えます。
- HEART: 被弾を1回無効化します。累積します。

### 単発系

- PUNCH: 歩いて爆弾に触れると、爆弾を進行方向へ1マス押せます。
- GLOVE: `E` で隣の爆弾を進行方向へ最大3マス投げます。
- WALL PASS: 通常状態の壊せる壁を通過できます。燃えている壁は通過できません。
- PIERCE BOMB: 爆風が壊せる壁を破壊しながら貫通します。
- REMOTE: 設置した爆弾が自動爆発せず、`E` で古い順に起爆します。

PUNCH / GLOVE / WALL PASS / PIERCE BOMB / REMOTE は死亡すると失われ、以後のショップ抽選に再び出ます。

## コスト

- BOMB / FIRE / SPEED はレベルが上がるほど必要SCOREが増えます。
- BOMB +2 / FIRE +2 / SPEED +2 は、通常2レベル分の合算コストに追加で500 SCOREが必要です。
- BOMB +3 / FIRE +3 / SPEED +3 は、通常3レベル分の合算コストに追加で1500 SCOREが必要です。
- 基本コストは BOMB `500 + 現在Lv * 300`、FIRE `600 + 現在Lv * 350`、SPEED `450 + 現在Lv * 250` です。
- 単発系の基本コストは PUNCH 2200、GLOVE 2400、WALL PASS 2600、PIERCE BOMB 2800、REMOTE 3000 です。

## デバッグ

`Ctrl + Alt + Shift` でデバッグコンソールを開けます。

調整できる項目:

- STAGE
- SCORE
- LIFE
- HEART
- BOMB LIMIT
- FIRE RANGE
- SPEED LEVEL
- PUNCH
- GLOVE
- WALL PASS
- PIERCE BOMB
- REMOTE

`APPLY STAGE` で指定ステージへ移動し、`RESET POWER` で強化状態を初期化できます。

## アセット

- `BGM/OP_BGM.mp3`: OP BGM
- `BGM/Explosion.mp3`: 爆発音
- `OPimage/`: OP画像
- `assets/enemies/jksoldier-walk.png`: JKsoldierの元スプライトシート
- `jksoldier-sprite-data.js`: JKsoldierスプライトシートのdata URI版。ブラウザ上で確実に読み込むため、`game.js`より先に読み込みます。
- `jksoldier-preview.html`: JKsoldier歩行アニメーション確認用の単独プレビュー

追加のタイトルBGM、ステージ開始BGM、エリア別BGMはローカル検証用です。public公開版には権利確認済みの音声だけを含めます。

## ローカル起動

静的ファイルだけで動作します。簡易サーバーを起動して `index.html` を開いてください。

```powershell
python -m http.server 8000
```

ブラウザで <http://localhost:8000> を開きます。

OPを飛ばして直接ゲームを起動したい場合は、次のURLを使えます。

```text
http://localhost:8000/index.html?skipOp=1
```

JKsoldierのゲーム内プレビューを出したい場合は、次のURLを使えます。

```text
http://localhost:8000/index.html?skipOp=1&jkPreview=1
```
