# Super Komatsu Man

Phaser 3 で作った、一人用のボンバーマン風アクションゲームです。

## 遊び方

- 移動: `WASD` / 矢印キー
- 爆弾設置: `Space`
- アクション: `E` (グローブで爆弾投げ / リモコン爆弾の起爆)
- リスタート: `R`
- 一時停止: `P`
- デバッグコンソール: `Ctrl + Alt + Shift` 同時押し。ステージ、SCORE、LIFE、HEART、BOMB/FIRE/SPEED、単発系アイテムを調整できます。

スマートフォンでは画面下の方向ボタンと `BOMB` ボタンを使えます。

敵を全滅させ、壊せる壁に隠された出口へ入るとステージクリアです。全20ステージあります。
5エリア構成で、各エリアに4ステージあります。
敵を倒すと獲得SCOREがその場に表示されます。短時間に同時撃破すると、2体目は1.5倍、3体目は2倍、以降は0.5倍ずつ倍率が上がります。

| AREA | STAGE | イメージ |
| --- | --- | --- |
| 1 STANDARD | 01〜04 | スタンダードな青いグリッド |
| 2 COAST | 05〜08 | 海岸。破壊可能ブロックは貝殻、破壊不能ブロックは砂の城 |
| 3 STEEL CITY | 09〜12 | 鋼鉄の街。破壊可能ブロックは家、破壊不能ブロックはビル |
| 4 FOREST | 13〜16 | 森。破壊可能ブロックは切り株、破壊不能ブロックは岩 |
| 5 SPACESHIP | 17〜20 | 宇宙船。破壊可能ブロックは基盤、破壊不能ブロックは鉄板 |

ステージクリア後は、SCOREを使って抽選された3つのパワーアップから1つを選択できます。
購入せずに次へ進むこともできます。

## パワーアップ

| レア度 | 出現率 | パワーアップ |
| --- | ---: | --- |
| COMMON | 70% | BOMB +1、FIRE +1、SPEED +1、LIFE +1 |
| RARE | 25% | BOMB +2、FIRE +2、SPEED +2、HEART、LIFE +2 |
| EPIC | 5% | BOMB +3、FIRE +3、SPEED +3、PUNCH、GLOVE、WALL PASS、PIERCE BOMB、REMOTE、LIFE +3 |

- BOMB / FIRE / SPEED の +2 は通常2レベル分の合算コストに追加で500 SCOREが必要です。
- BOMB / FIRE / SPEED の +3 は通常3レベル分の合算コストに追加で1500 SCOREが必要です。
- BOMB / FIRE / SPEED はレベルが上がるほど必要SCOREが増えます。
- 最大値は BOMB 同時設置数8、FIRE 爆風距離8、SPEED 強化段階8です。
- 基本コストは BOMB `500 + 現在Lv * 300`、FIRE `600 + 現在Lv * 350`、SPEED `450 + 現在Lv * 250` です。
- PUNCH / GLOVE / WALL PASS / PIERCE BOMB / REMOTE は単発系です。死亡すると失われ、以後のショップ抽選に再び出ます。
- PUNCH: 歩いて爆弾に触れると、爆弾を進行方向へ1マス押せます。
- GLOVE: `E` で隣の爆弾を進行方向へ最大3マス投げます。
- WALL PASS: 通常状態の壊せる壁を通過できます。燃えている壁は通過できません。
- PIERCE BOMB: 爆風が壊せる壁を破壊しながら貫通します。
- REMOTE: 設置した爆弾が自動爆発せず、`E` で古い順に起爆します。
- HEART: 被弾を1回無効化します。累積できます。
- LIFE +1〜+3: 残機が即時増えます。

## 敵リスト

| 敵 | LIFE | 行動 |
| --- | ---: | --- |
| ノーマル | 1 | 壁や爆弾にぶつかるまで直進し、ぶつかると方向転換します。 |
| チェイサー | 1 | マス中央で進路を判断し、プレイヤーへ近づく方向を選びます。 |
| ゴースト | 1 | 壊せる壁を通過できます。移動方向はランダムで、壁抜け中に詰まった場合は自動復帰します。 |
| パックンマン | 2 | 青いパックマン風の敵です。移動方向へ口を向け、パクパクしながら移動します。普段はランダム移動し、2〜3回交差点を通るごとに曲がります。爆弾を見つけると近づき、接触した爆弾を消します。 |
| シュバリエ | 2 | 騎士風の敵です。前方2マス以内にプレイヤーがいると、槍を表示して速度2倍で突進します。 |
| バクダソ | 1 | 爆弾型の敵です。約7秒ごとに爆弾化して爆発し、その後元に戻ります。爆弾化中は爆風で誘爆しますが倒れません。 |
| ケダマ | 1 | 爆風を受けると燃え上がって高速追跡を始め、約5秒後に爆発します。燃焼中は倒せません。 |
| テポドン | 1 | 前方にプレイヤーを見つけると、ロケット形態で高速突進します。壁にぶつかるかプレイヤーへ到達すると広範囲に爆発します。 |

## ローカル起動

静的ファイルだけで動作します。簡易サーバーを起動して `index.html` を開いてください。

```powershell
python -m http.server 8000
```

ブラウザで <http://localhost:8000> を開きます。

## GitHub Pages

1. このフォルダを GitHub リポジトリへ push します。
2. リポジトリの `Settings > Pages` を開きます。
3. `Build and deployment` の Source に `GitHub Actions` を選びます。

`main` ブランチへの push 時に `.github/workflows/pages.yml` がサイトを公開します。
